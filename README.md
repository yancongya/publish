# Quick Cartilage Rigging Blender 插件

一个用于快速创建软骨绑定的Blender插件。

## 📚 完整文档索引

欢迎使用"快速软骨绑定"插件！本插件旨在简化和加速Blender中软骨链（如尾巴、触手、飘带等）的绑定流程。

## 插件功能

- 骨骼细分（平均细分和斐波那契细分）
- FK控制器绑定
- 软骨效果（阻尼追踪）
- 实时参数调节
- 右键菜单快捷操作
- 骨骼集合管理

## 支持的Blender版本

- 4.5.0 及以上版本
